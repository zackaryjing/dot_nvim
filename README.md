# Neovim 配置

基于 [LazyVim](https://github.com/LazyVim/LazyVim) 的个人 Neovim 配置。

## 安装

确保已安装 Neovim 和 Git，然后执行：

```bash
git clone git@github.com:zackaryjing/dot_nvim.git ~/.config/nvim
nvim
```

首次启动时，LazyVim 会自动安装插件。

新机器的环境准备见 [Neovim 前置环境](docs/prerequisites.md)。
使用说明见 [个人速查表](docs/cheatsheet.md)。更多配置方式参考
[LazyVim 官方文档](https://lazyvim.github.io/)。
