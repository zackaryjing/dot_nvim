# Neovim 个人速查表

## 消息

| 快捷键或命令 | 用途 |
| --- | --- |
| `<leader>snh` | 查看 Noice 消息历史 |
| `<leader>snl` | 查看最后一条 Noice 消息 |
| `<leader>sna` | 查看全部 Noice 消息 |
| `<leader>n` | 查看 Snacks 通知历史 |
| `<leader>un` | 关闭浮动通知 |
| `:messages` | 查看 Neovim 原生消息历史 |

`<leader>` 默认为空格键。

## LSP

| 快捷键 | 用途 |
| --- | --- |
| `gd` | 跳转到定义 |
| `gD` | 跳转到声明 |
| `gr` | 查看引用 |
| `gI` | 跳转到实现 |
| `gy` | 跳转到类型定义 |
| `K` | 显示悬浮文档 |
| `<leader>cl` | 查看当前 LSP 信息 |

## Python

| 快捷键 | 用途 |
| --- | --- |
| `<leader>cv` | 为当前项目选择 Python 虚拟环境 |

## 诊断

| 快捷键 | 用途 |
| --- | --- |
| `<leader>cd` | 在浮窗中显示当前行的完整诊断 |
| `]d` / `[d` | 跳转到下一个 / 上一个诊断 |
| `]e` / `[e` | 跳转到下一个 / 上一个错误 |
| `]w` / `[w` | 跳转到下一个 / 上一个警告 |
| `<leader>xx` | 使用 Trouble 查看整个项目的诊断列表 |
| `<leader>xX` | 使用 Trouble 查看当前文件的诊断列表 |
